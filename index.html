Image Target AR Video

Load a .mind target + video, then scan the image. Video autoplays and loops.
You need a compiled MindAR image target file. This is the image-tracking file, not JPG/PNG.
Use MP4 H.264 if possible. The app forces muted autoplay + loop for mobile browsers.
For landscape video use 1 × 0.5625. For square use 1 × 1. For portrait use 0.5625 × 1.
Choose a .mind target and a video.
Important for GitHub Pages
Upload this index.html directly at the repository root. No folders needed. Open the GitHub Pages HTTPS link on mobile. Camera will not work from a normal local file path.

Tap to enable video

Some phones need one tap before video texture can play.
