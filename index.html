No-server AR link builder

Paste direct URLs for your compiled .mind target and AR asset. This page generates a viewer link you can put behind a QR code.

Works best when files are hosted on GitHub Pages, Netlify, Cloudinary, Firebase Storage, or any host that allows direct HTTPS + CORS.

Create this file with the MindAR Image Target Compiler, then host it online.

Open viewer

Workflow

  1. Choose a high-detail poster/image as your AR marker.
  2. Compile it into targets.mind using MindAR Image Target Compiler.
  3. Host targets.mind and your AR asset online as direct HTTPS links.
  4. Paste links here, generate the viewer URL, then create a QR code from it.
