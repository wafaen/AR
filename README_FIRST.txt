AR VIDEO BUILDER - FAST USE

1. Open index.html in Chrome/Edge on your computer.
2. Click + New project.
3. Add your target image/poster.
4. Add your MP4 video.
5. Click Compile .mind target. This needs internet because the compiler loads from CDN.
6. Click Export AR project ZIP.
7. Unzip the exported project and upload the whole folder to an HTTPS host.
8. Open the uploaded URL on a phone, allow camera, point at the image.

Important:
- Video is muted, autoplayed, looped, and placed full-frame by default.
- No VR button is enabled.
- The exported viewer uses browser camera, so local file URLs will not work on phones. Use HTTPS.
- Good image targets need detail/texture. Flat posters with large blank areas track badly.
- For iPhone, the first Start AR tap is required by the browser. After target detection the video starts.

Free hosting options:
- Netlify Drop
- Vercel
